====================================================
Gloria Trading 車両管理画面 - 使い方
====================================================

【アクセス方法】
ブラウザで以下のURLを開いてください：
  https://あなたのドメイン/admin/

【初期パスワード】
  gloria2025

【パスワードの変更方法】
  admin/index.php をテキストエディタで開き、
  以下の行のパスワードを変更してください：
  define('ADMIN_PASSWORD', 'gloria2025');

【できること】
  1. 車両の追加・編集・削除
  2. 画像のアップロード（複数枚対応）
  3. 画像の削除・並び替え
  4. 保存すると自動的にサイトに反映

【注意事項】
  - images/vehicles/ フォルダに書き込み権限が必要です
    (パーミッション: 755 または 777)
  - data/vehicles.json に書き込み権限が必要です
    (パーミッション: 644 または 666)
  - さくらのレンタルサーバーの場合、FTPで
    上記ファイルのパーミッションを確認してください

【ファイル構成】
  admin/
    index.php  - 車両一覧・ログイン画面
    edit.php   - 車両追加・編集画面
    .htaccess  - セキュリティ設定
